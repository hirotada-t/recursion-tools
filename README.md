# recursion-tools

## Tamepermonkey-shortcut.user.js

[Recursion](https://recursionist.io/)でテスト・投稿をショートカットキーで操作するためのスクリプトです。

動作|コマンド
--|--
テスト | Command + Shift + s
実行 | Command + Shift + x

### 動作環境

- Chrome ブラウザ + Mac での挙動を確認しています
